# 🎯 Quiz Master
A terminal-based quiz application built in C as a group project 
during B.Tech 1st Year at Jaypee Institute of Technology.

## 👥 Team
- Kishita Gupta
- Shreya Agarwal
- Idika Kumar

## ✨ Features
- 👤 User registration & login with password protection
- 📚 5 subjects — Physics, Mathematics, History, Sports, 
  Computer Science
- 🎯 3 difficulty levels — Easy, Medium, Hard
- ⏱️ Timed questions (15 seconds per question)
- 📊 Score tracking, quiz history & accuracy stats
- 💾 Persistent data storage using binary file I/O
- 🖥️ Cross-platform support (Windows & Linux)

## 🚀 How to Run

### Windows
```bash
gcc quiz_master.c -o quiz_master.exe
quiz_master.exe
```

### Linux / Mac
```bash
gcc quiz_master.c -o quiz_master
./quiz_master
```

## 🛠️ Tech Stack
- Language: C
- Concepts: Structs, File I/O, Arrays, Functions, 
  Conditional compilation, Cross-platform development

## 🖼️ How It Works
1. Sign up or log in with your username & password
2. Choose a subject and difficulty level
3. Answer 5 timed questions (15 sec each)
4. View your score, accuracy & quiz history anytime
